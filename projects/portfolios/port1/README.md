# 💼 Portfolio Page

**Category:** HTML Fundamentals  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/html-fundamentals/Portfolio/index.html)

## What is it?
A simple personal portfolio page — an early practice at building a structured personal site with navigation and content sections.

## Concepts Covered
- `<nav>` with anchor links for in-page navigation
- `<header>`, `<main>`, `<footer>` structure
- Project card layout with Flexbox
- Inline vs block element behaviour
- Hover states with CSS transitions
