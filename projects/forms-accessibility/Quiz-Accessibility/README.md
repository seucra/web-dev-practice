# ♿ Quiz (Accessibility)

**Category:** Forms & Accessibility  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/forms-accessibility/Quiz-Accessibility/index.html)

## What is it?
An accessible form-based quiz with full keyboard navigation, ARIA attributes, and semantic structure.

## Concepts Covered
- `role` attributes (`region`, `listitem`)
- `aria-labelledby` and `aria-label`
- Accessible `<nav>` with jump links
- Proper `<label>` associations (`for` + `id`)
- Tab order and keyboard focus management
- `<section>` with meaningful headings for screen readers
