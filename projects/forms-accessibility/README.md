## Skills Demonstrated
- Semantic form elements: `<form>`, `<fieldset>`, `<legend>`, `<label>`
- Input types: `text`, `email`, `number`, `radio`, `checkbox`, `select`, `textarea`
- HTML validation attributes: `required`, `minlength`, `pattern`
- ARIA roles and attributes for screen reader support
- Keyboard navigation and focus management
