# 📋 Survey Form

**Category:** Forms & Accessibility  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/forms-accessibility/SurveyForm/index.html)

## What is it?
A complete, styled survey form — a freeCodeCamp certification project demonstrating diverse input types in a real-world context.

## Concepts Covered
- Full form semantics: `<form>`, `<label>`, `<input>`, `<select>`, `<textarea>`
- Radio button groups with `name` attributes
- Checkbox groups
- `<option>` with default placeholder (`value=""`)
- Styled form layout with consistent spacing
- `id`/`for` label pairing for accessibility
