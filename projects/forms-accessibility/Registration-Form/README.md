# 📝 Registration Form

**Category:** Forms & Accessibility  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/forms-accessibility/Registration-Form/index.html)

## What is it?
A styled user registration form covering the full range of HTML input types with validation.

## Concepts Covered
- Input types: `text`, `email`, `password`, `number`, `file`, `radio`, `checkbox`
- `<fieldset>` and `<legend>` for grouping
- `<select>` and `<option>` dropdowns
- `<textarea>` for multi-line input
- HTML attributes: `required`, `minlength`, `maxlength`, `pattern`
- CSS styling of form elements including custom checkboxes
