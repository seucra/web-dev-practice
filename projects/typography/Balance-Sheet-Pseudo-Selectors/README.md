# 📊 Balance Sheet

**Category:** Typography  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/typography/Balance-Sheet-Pseudo-Selectors/index.html)

## What is it?
A styled financial balance sheet showcasing how CSS pseudo-selectors can target specific table rows and cells without adding extra classes.

## Concepts Covered
- `:nth-of-type()` and `:last-of-type` pseudo-selectors
- `::before` and `::after` pseudo-elements
- `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`
- `clip-path` for accessible text hiding
- Hover highlights on table rows
