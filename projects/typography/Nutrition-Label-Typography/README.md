# 🥗 Nutrition Label

**Category:** Typography  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/typography/Nutrition-Label-Typography/index.html)

## What is it?
A pixel-perfect replica of a US FDA nutrition facts label — an advanced typography exercise.

## Concepts Covered
- Heavy use of `font-weight`, `font-size`, `line-height`
- `border` widths (thin, medium, thick)
- Flexbox for side-by-side label values
- Indented sub-items using `padding-left`
- `letter-spacing` and `text-transform` for label text
- Dividers using `border-bottom` with varying thickness
