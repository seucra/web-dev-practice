# ☕ Cafe Menu

**Category:** Typography  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/typography/Cafe-Menu/index.html)

## What is it?
A warm, inviting cafe menu layout — practice with web fonts, typographic rhythm, and image placement.

## Concepts Covered
- Google Fonts integration via `<link>`
- `font-family`, `font-size`, `font-style`
- `text-align` and `letter-spacing`
- Two-column layout using Flexbox (`justify-content: space-between`)
- `max-width` with `margin: auto` for centred content
- Background image with fallback colour
