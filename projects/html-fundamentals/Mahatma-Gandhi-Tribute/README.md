# 🕊️ Mahatma Gandhi Tribute

**Category:** HTML Fundamentals  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/html-fundamentals/Mahatma-Gandhi-Tribute/index.html)

## What is it?
A respectfully designed tribute page honouring Mahatma Gandhi, built as a practice project with semantic HTML and basic CSS styling.

## Concepts Covered
- `<figure>` and `<figcaption>` for captioned images
- `<blockquote>` and `<cite>` for quotations
- `<article>` as a self-contained unit
- Clean typographic styling with system fonts
- External links with `target="_blank"` and `rel="noopener"`
