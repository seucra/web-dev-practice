# 🧠 Human Brain

**Category:** HTML Fundamentals  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/html-fundamentals/Human-Brain/index.html)

## What is it?
An HTML document structured around human brain anatomy — an exercise in semantic document outlining.

## Concepts Covered
- Proper heading hierarchy (`h1` → `h3`)
- `<article>`, `<section>`, `<aside>`, `<nav>`
- Descriptive lists (`<dl>`, `<dt>`, `<dd>`)
- Academic-style content structure
