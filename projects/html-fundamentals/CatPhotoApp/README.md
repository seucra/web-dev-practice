# 📷 Cat Photo App

**Category:** HTML Fundamentals  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/html-fundamentals/CatPhotoApp/index.html)

## What is it?
My very first freeCodeCamp project — a simple webpage about cats built entirely in HTML.

## Concepts Covered
- `<img>` tags and `alt` attributes
- Hyperlinks (`<a href>`)
- Unordered + ordered lists
- `<figure>` and `<figcaption>`
- Basic HTML form elements (`<input>`, `<button>`, radio buttons, checkboxes)
- `<section>` and semantic structure
