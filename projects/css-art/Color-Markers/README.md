# 🖍️ Color Markers

**Category:** CSS Art  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/css-art/Color-Markers/index.html)

## What is it?
Three marker pen illustrations demonstrating how gradients and shadows stack to create depth and shine.

## Concepts Covered
- `linear-gradient()` with multiple colour stops
- `box-shadow` (multiple layers, inset vs outer)
- RGB, HSL, and hex colour models
- Understanding colour contrast and blending
