# 🐱 Cat Painting

**Category:** CSS Art  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/css-art/Cat-Painting-Intermediate-CSS/index.html)

## What is it?
A CSS-only cat portrait assembled from plain `<div>` elements — no images, just shapes.

## Concepts Covered
- Intermediate CSS selectors
- `border-radius` for organic shapes
- `position: absolute/relative` layering
- Precise sizing with `width`/`height`
- `z-index` for depth ordering
