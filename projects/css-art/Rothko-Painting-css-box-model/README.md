# 🎨 Rothko Painting

**Category:** CSS Art  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/css-art/Rothko-Painting-css-box-model/index.html)

## What is it?
Abstract colour-field art inspired by Mark Rothko's paintings — made with plain `<div>` elements and CSS.

## Concepts Covered
- Box model mastery: `margin`, `padding`, `border`
- `width` and `height` in pixels
- `filter: blur()` for soft edges
- `border-radius` for organic shapes
- Background colours and subtle shadows to create painted texture
