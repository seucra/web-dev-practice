# 🌆 City Skyline

**Category:** CSS Art  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/css-art/CitySkylines-Variables/index.html)

## What is it?
A vector-style city skyline built entirely with CSS — day and night modes driven by custom properties.

## Concepts Covered
- CSS custom properties (`--variable-name`)
- Reusing variables for consistent theming
- Layered `div` elements to build architecture
- `linear-gradient` for sky backgrounds
- Responsive scaling with `%` units
