# 🎡 Ferris Wheel

**Category:** CSS Art  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/css-art/FerrisWheel-Animation/index.html)

## What is it?
A fully animated Ferris wheel with spinning spokes and swinging cabins — powered entirely by CSS, zero JavaScript.

## Concepts Covered
- CSS `@keyframes` animation
- `animation-duration`, `animation-timing-function`, `animation-iteration-count`
- `transform-origin` to set rotation pivot point
- Counter-rotation trick (cabins stay upright while wheel spins)
- `position: absolute` within a relative container
