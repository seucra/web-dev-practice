# 🖼️ Photo Gallery

**Category:** Layouts  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/layouts/Photo-Gallery-Flexbox/index.html)

## What is it?
A fluid responsive photo grid that rearranges itself at different viewport widths using Flexbox.

## Concepts Covered
- `display: flex` and `flex-wrap: wrap`
- `gap` for consistent spacing
- `flex-basis` and `min-width` for responsive sizing
- `object-fit: cover` for image cropping
- Responsive design without media queries (fluid Flexbox approach)
