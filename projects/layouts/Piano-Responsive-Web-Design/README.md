# 🎹 Piano

**Category:** Layouts  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/layouts/Piano-Responsive-Web-Design/index.html)

## What is it?
A CSS piano keyboard that adapts its size at different screen widths — practice with responsive design principles.

## Concepts Covered
- Flexbox for side-by-side key arrangement
- `position: absolute/relative` for black key placement
- `@media` queries for responsive breakpoints
- `max-width` and viewport-relative sizing
