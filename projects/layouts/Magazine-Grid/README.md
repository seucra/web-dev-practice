# 📰 Magazine Grid

**Category:** Layouts  
**Live demo:** [Open ↗](https://seucra.github.io/web-dev-practice/projects/layouts/Magazine-Grid/index.html)

## What is it?
A multi-column editorial-style layout mimicking a print magazine — built with CSS Grid.

## Concepts Covered
- `grid-template-columns` and `grid-template-rows`
- Named grid areas with `grid-template-areas`
- `grid-column` / `grid-row` spanning
- `column-gap` and `row-gap`
- Placing images and text blocks within a grid
